WebRTC ScreenShare
===================

Description:
-------------
WebRTC ScreenShare is a real-time collaboration platform that allows users to share their screens and stream video seamlessly. It includes features like chat, audio control, and remote management for enhanced online meetings.

https://github.com/user-attachments/assets/150b6632-b1ae-4327-89b8-35771b7646d5

Features:
---------
- Screen sharing
- Video streaming
- Chat functionality
- Remote audio control
- Volume adjustment

Requirements:
-------------
- Node.js
- Socket.io

Installation:
-------------
1. Clone the repository:
   git clone https://github.com/yourusername/webrtc-screenshare.git
2. Navigate to the project directory:
   cd webrtc-screenshare
3. Install dependencies:
   npm install

Usage:
------
1. Start the server:
   node server.js
2. Open your browser and navigate to:
   http://localhost:3000
3. Enter a room name and click "Join Room" to start sharing your screen.
